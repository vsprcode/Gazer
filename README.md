# GAZER.

**NEURAL OBSERVATION INTERFACE** — Retro-futuristic RSS reader prototype.

> Monitor feeds & intel streams across all active nodes.

## Stack
- Single-file HTML + CSS + JS prototype
- Mobile-first (480px), Space Mono font
- Web Share API for sharing articles
- Design inspired by brutalist / retro-OS aesthetics

## Screens
- **DASHBOARD** — Stats & entry point
- **LOGS** — Signal feed with priority cards
- **NODES** — RSS source management  
- **SETTINGS** — System configuration
